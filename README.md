# Shamir's Secret Sharing (SSS) Solver

A secure, web-based tool designed to reconstruct secrets from shared data points using the **Shamir's Secret Sharing Algorithm**. This project utilizes **Lagrange Interpolation** to process polynomial-based cryptography through a modern, professional interface.
Live Link : https://malarkoditt.github.io/Shamir-s-secret-share/
 ## 📖 How to Use

1. **Prepare your JSON:** Ensure your input follows the standard SSS format (keys for `n` and `k`, followed by share objects with `base` and `value`).
2. **Upload:** Open the website and upload your JSON file via the "Initialize Data Stream" portal.
3. **Reconstruct:** The system will automatically decode the shares and display the **Reconstructed Secret Key** instantly.
